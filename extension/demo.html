🎭 Emoji Manager Browser Extension

Advanced emoji management with Vue 3 + Vite + TailwindCSS + Ant Design Vue

✨ Key Features

📏 Image Scale Control

Adjust output image size from 5% to 150% with real-time preview

🎯 Popup Interface

Compact popup with emoji grid, settings navigation, and secondary menu

⚙️ Options Management

Full-featured settings page for emoji group and content management

🔄 Drag & Drop

Intuitive reordering of emojis within groups and between groups

📁 Group Management

Create, edit, and organize emoji groups with custom icons

☁️ Configuration Sync

Export/import configurations and cloud synchronization

🏗️ Technical Implementation

🔧 Modern Tech Stack

Vue 3 Composition API + Vite 6 + TailwindCSS 3 + Ant Design Vue 4

📦 Build Optimization

Code splitting, vendor chunking, and modern JavaScript output

🏪 State Management

Pinia for reactive state with Chrome storage API integration

🛡️ Extension Standards

Manifest V3 compliant with proper permissions and CSP

📋 Project Structure

extension/
├── src/
│   ├── popup/           # Popup window components
│   ├── options/         # Settings page components  
│   ├── stores/          # Pinia state management
│   ├── styles/          # TailwindCSS styles
│   └── components/      # Shared components
├── public/
│   ├── manifest.json    # Extension manifest
│   └── icons/           # Extension icons
└── dist/                # Built extension files
